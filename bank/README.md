# 🏦 Banking System (Java)
# 📌 Overview

This project is a simple Banking System developed in Java that simulates basic banking operations such as account creation, deposit, withdrawal, and balance inquiry. It is designed to demonstrate object-oriented programming concepts and file handling in Java.

# 🚀 Features
🧾 Create a new bank account
💰 Deposit money
💸 Withdraw money
📊 Check account balance
🔐 Basic validation for transactions
📁 File handling for data persistence (optional)
# 🛠️ Technologies Used
Java (Core Java)
OOP Concepts (Classes, Objects, Inheritance, Encapsulation)
File Handling (if implemented)
# 📂 Project Structure
The project is organized into several packages and directories:

    bin: Contains compiled Java bytecode files.

    src: Contains the source code files, organized into packages:
        Bank: Core banking classes.
        Data: Data-related operations.
        Exceptions: Handling of exceptions.
        GUI: Graphical User Interface components.
        img: Image resources for the GUI.

# ⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/srishtiseema/banking-system.git
Navigate to the project directory:
cd banking-system
Compile the Java files:
javac src/*.java
Run the program:
java src.Main
# 🧠 How It Works
The system allows users to interact via a console-based menu.
Each account is represented as an object.
Transactions update the account balance accordingly.
Data can be stored in a file to maintain records.
